# 🚀 Space-Themed Birthday Invitation Website

A premium, interactive birthday invitation website built as part of the **Codebasics AI Pro Program**.

This project explores how AI orchestration can accelerate product development by combining multiple AI tools across planning, design, and implementation.

---

## 🌟 Project Overview

The website delivers an immersive, space-themed invitation experience featuring:

- Personalized guest onboarding
- Dynamic countdown timer
- Interactive RSVP flow
- Event timeline
- Venue showcase
- Google Maps integration
- AI-generated visuals
- Smooth animations
- Fully responsive design

---

## 🎯 Objective

The goal of this project was not only to build a birthday invitation website but also to experiment with an AI-first product development workflow.

Instead of relying on a single AI tool, different tools were leveraged at different stages of the project lifecycle to maximize efficiency and quality.

---

## 🤖 AI Workflow

### 📋 Claude

- Generated the Product Requirements Document (PRD)
- Defined feature requirements
- Structured the overall user experience
- Helped refine product specifications

### 🎨 ChatGPT

- Generated and refined visual assets
- Assisted with UI/UX improvements
- Helped iterate on design concepts
- Refined content and user interactions

### ⚡ Antigravity

- Transformed requirements and assets into a fully functional website
- Accelerated implementation
- Enabled rapid iteration and refinement

---

## 💡 Key Learning

This project reinforced the importance of **AI orchestration**.

The quality of the final outcome depended not on a single AI model, but on how effectively context, requirements, and outputs flowed between multiple AI systems.

As AI capabilities continue to evolve, workflow design is becoming just as important as model selection.
